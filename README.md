Sistema de Agendamento MobileEste é um aplicativo desenvolvido em React Native (Expo) focado em facilitar o agendamento de serviços diversos, como barbearia, clínicas, pet shops, entre outros.

👥
Integrantes: \* Luiz Felipe de Oliveira Marques

📝
Descrição do ProjetoO projeto consiste em uma plataforma de agendamento onde usuários podem explorar diferentes estabelecimentos, verificar serviços disponíveis e realizar reservas de horários de forma prática e rápida.O foco é a versatilidade, permitindo o uso em nichos variados:Saúde: Médicos, Dentistas, Nutricionistas.Estética: Barbearia, Manicure, Pilates.Serviços: Lava-car, Pet Shop.

📱
Telas do AplicativoCom base na estrutura de arquivos em src/screens, o aplicativo conta com as seguintes interfaces:TelaDescriçãoSplashTela de inicialização com a identidade visual do app.LoginAcesso seguro para o usuário gerenciar suas reservas.HomePainel principal com destaques e categorias de serviços.ExplorarFerramenta de busca e descoberta de novos estabelecimentos.EstabelecimentoDetalhes sobre o local, fotos, descrição e lista de serviços oferecidos.Fazer ReservaFluxo para seleção de data, horário e confirmação do agendamento.ReservasListagem de todos os agendamentos realizados pelo usuário (histórico e próximos).PerfilGestão de dados pessoais e configurações da conta.🛠️ Tecnologias UtilizadasReact NativeExpoJavaScript
